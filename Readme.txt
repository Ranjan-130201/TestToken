This Project is to practice ERC20 token 
A contract TestToken (ERC20) is deployed on Goerli TestNet
The Website is built on react
It interacts with contract TestToken to mint, transfer and display balance of the account
Add token Button adds the tiken to metamask account
Transfer button transfers the Tokens
Mint gives you 20 TestTokens 
Transaction information is displayed after the transferis complete
Transaction information is then sent to another contract TransactionReciept
which stores the Transaction information
The Transactionhash of this is displayed
